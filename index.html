Features

  • Unlimited number of image-slides
  • Slide expanding is animated
  • Written with SASS and Compass
  • Easily customizable through SASS variables
  • No JavaScript, just CSS
  • Support for several differently configured sliders in one page
  • * Title transformations and animations require browser support.
  • ** I don't care about old IE-s. Neither should you or your clients.

Notes

  • Untested
  • Specify width and height tags on your img elements.

Usage

Note: Use Compass to recompile: change scss config files in scss/*.scss to suit your needs and run compass watch in the root directory. Change config.rb for compass settings.

  1. Open scss/example1.scss and customize.
  2. Include the compiled stylesheet.
    <link rel="stylesheet" href="css/example1.css">
  3. Specify slides in your HTML
<ul class="front-slides">
    <li class="slide-1">
        <a href="#">
            <img src="http://placekitten.com/1000/800" height="800" width="1000" alt="Slide 1"/>
            <h4>Dana</h4>
        </a>
    </li>
    <li class="slide-2">
        <a href="#">
            <img src="http://placekitten.com/800/1000" height="1000" width="800" alt="Slide 2"/>
            <h4>Mrs Pouty</h4>
        </a>
    </li>
</ul>

Numeric slide classes are used for positioning the images.
